# MindS\*\*t

:: Written in Python

A programming language that compiles into the notoriously simple yet difficult esoteric language [BrainF\*\*k](https://esolangs.org/wiki/Brainfuck). Code is intended to stylistically replicate other common programming languages such as C and Rust, but is customized for generating BrainF\*\*k code.
